# Code of Conduct

AgentPower contributors are expected to keep discussions focused, respectful, and useful.

## Expected Behavior

- Assume good intent.
- Critique code and product decisions, not people.
- Keep security and payment discussions precise.
- Respect maintainer decisions around risk and release readiness.

## Unacceptable Behavior

- Harassment or personal attacks.
- Publishing secrets or exploit details.
- Pressuring maintainers to bypass review gates.
- Misrepresenting mock payment or agent behavior as production-ready.
