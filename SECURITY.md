# Security Policy

AgentPower currently runs as a local MVP with mock GitHub, payment, queue, and sandbox adapters.

## Sensitive Areas

Human review is required for changes touching:

- payment gateway credentials,
- billing and donations,
- authentication,
- authorization and permissions,
- production secrets,
- deployment credentials,
- encryption,
- database migrations,
- security policy,
- autonomous merge rules.

## Reporting

For now, report security issues privately to the maintainers of the repository where this project is published.

Do not publish exploitable details until maintainers have acknowledged and triaged the issue.
